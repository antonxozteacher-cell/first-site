Мини-методичка по HTML


Абзац

Тег <p> создаёт абзац.

Синтаксис:

<p> … </p>

Пример:

Это абзац текста.

Заголовок (h1…h6)

Теги заголовков: <h1> самый большой, <h6> самый маленький.

Синтаксис:

<h1> … </h1>
<h2> … </h2>
...
<h6> … </h6>

Пример:

Пример h3

Пример h5

Жирный

Тег <b> делает текст жирным.

Синтаксис:

<b> … </b>

Пример:

Обычный текст и жирный текст.
